Questions

  1. What does the server in the client server architecture do?
    The server contains the files. And when requested, sends them out to the client.
  2. What does the client do?
    The client requests and receives the files from the server, then presents them to the user if needed.
  3. What is an IP address?
    An IP (Internet Protocol) address is used for identification and addressing of different computers on a network.
  4. What is DNS?
    A DNS (Domain Name System) is like a phonebook for the internet, linking domain names to IP addresses.
  5. What is the difference between the Internet and WWW?
    The internet is the infrastructure. The www (world wide web) is like a service containing information, displayed through the internet framework.
  6. Explain what HTML is
    HTML is a markup text language used for making websites. It also supports hypertext which is used to link webpages together.
